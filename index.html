Welcome to Orthodox Tewahedo Misbak Online!

How to use this:

Click on the month that you will see below and a page with a player will pop up. Then, click on play if it doesn't automatically start playing from the start. To pick a specific date, click on this button.

 

Note: Some months are not available. If you have audios, please submit them by creating a pull request on this Github repository
